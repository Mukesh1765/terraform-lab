# Terraform Lab

This repository contains assignments demonstrating Infrastructure as Code using Terraform.

## Contents

- Day 1: Basic Terraform setup and local file creation
- Day 2: Variables and environment-based configurations
- Day 3: Docker container provisioning
- Day 4: Terraform modules and reusable infrastructure
- Day 5: Dynamic infrastructure creation using for_each

## Tools Used
- Terraform
- Docker
- GitHub Actions (CI)

## Outcome
This lab demonstrates automation, modular design, and scalable infrastructure using Terraform.
